# Phase 7 — Projects & Job Hunt (Month 8-12)

> **Goal:** Land a 20+ LPA AI Engineer role.
> **Time:** 2 hrs learning + 4-6 hrs applying/networking per day
> **Output:** Multiple offers, ideally competing

---

## 🎯 The job hunt isn't luck. It's a system.

Most people "apply on Naukri" and wonder why they get ghosted. You won't. You'll:
- Build a portfolio that gets found
- Network intentionally (50% of jobs come from referrals)
- Apply strategically (right companies, right roles)
- Interview with confidence (you've prepped)

---

## 📅 The 5 must-ship portfolio projects

You need 5 strong projects. Specs below:

### Project 1: Document Q&A with RAG (the classic)
- **Stack:** LangChain + OpenAI + Pinecone + Next.js + FastAPI
- **Features:** PDF upload, citations, streaming, conversation history
- **Eval:** RAGAS, 50-question test set
- **Deploy:** Vercel + Render + Pinecone
- **Why:** Most common AI Engineer interview question

### Project 2: AI Agent that does real work
- **Stack:** LangGraph + Claude/OpenAI + web search + code execution
- **Example ideas:** Research agent, code reviewer, data analyst
- **Eval:** Success rate, cost per task
- **Why:** Shows you can build agentic systems

### Project 3: Fine-tuned model for a domain
- **Stack:** Hugging Face + PEFT/QLoRA + Llama 3
- **Example ideas:** Legal Q&A, medical Q&A, customer support, code gen for a niche language
- **Why:** Shows you understand model training, not just API calls

### Project 4: Production ML pipeline
- **Stack:** MLflow + scikit-learn/XGBoost + FastAPI + Docker + AWS
- **Features:** Training pipeline, model registry, A/B testing, monitoring
- **Why:** Shows production engineering (your edge over pure ML folks)

### Project 5: Full-stack AI SaaS
- **Stack:** Next.js + Stripe + Supabase + OpenAI + background jobs
- **Features:** Auth, multi-tenant, payments, usage tracking, admin dashboard
- **Why:** Shows you can ship a real product. **This is what 25+ LPA candidates look like.**

## 📝 Resume that gets 20+ LPA

See **`career/resume-portfolio.md`** for the full guide. TL;DR:

- **Header:** Name, role-targeted title, location, links (GitHub, LinkedIn, portfolio)
- **Summary:** 3 lines, AI Engineer focused, mentions your LLM/RAG/agent projects
- **Skills:** Group by category, prioritize AI/LLM tools

## 🤝 The networking playbook

### What to post on LinkedIn (1x/week, minimum)

**Types of posts that work:**
1. **Project launch** — "I built X. Here's what I learned." with demo gif
2. **TIL (Today I Learned)** — "I learned X today. Here's a 5-min summary."
3. **Hot take** — "Unpopular opinion: [something about AI]"
4. **Comparison** — "I tried LangChain vs LlamaIndex. Here's what I found."
5. **Tutorial** — "How to build a RAG in 100 lines"

**Template that works:**
```
I [built/learned/discovered] [X].
Here's why it matters: [1 sentence].
Here's what I learned: [3 bullet points].
Here's the code/demo: [link]

#AI #LLM #MachineLearning
```

### DM templates that get responses

**To a senior AI engineer (not asking for a job):**
> Hi [Name], loved your post on [topic]. I've been building [X], and ran into [specific problem]. Mind if I ask how you'd approach it? Thanks!

**To a recruiter:**
> Hi [Name], I'm an AI Engineer specializing in LLM apps and RAG systems. I've shipped [X project, Y project]. Looking for senior AI Engineer roles in [city/remote]. Open to a 10-min chat if you're placing for such roles?

**To a hiring manager:**
> Hi [Name], saw your team is hiring for [role]. I just shipped [similar project] using [their stack]. Would love to chat about how I could contribute. Here's my GitHub: [link]

## 🎯 Interview prep

See **`career/interview-prep.md`** for the full guide. TL;DR:

**Round 1 — Coding (DSA):**
- 2 LeetCode Medium problems, 60 min
- Focus: arrays, strings, hash tables, trees, graphs, DP, sliding window
- 200 problems solved = 90% pass rate

**Round 2 — AI/LLM technical:**
- "How does RAG work? Walk me through it."
- "How would you evaluate a RAG system?"
- "How would you reduce LLM costs in production?"
- "Explain the transformer architecture."
- "What's the difference between fine-tuning and RAG? When to use which?"

**Round 3 — System design:**
- "Design a chatbot for [X] company"
- "Design a RAG system that handles 1M docs"
- "Design an agent for [task]"

**Round 4 — Culture fit / Behavioral:**
- "Tell me about a hard project."
- "Tell me about a time you disagreed with a teammate."
- "Why AI engineering?"

## 💰 Salary negotiation

See **`career/salary-negotiation.md`** for the full guide. TL;DR:

- **Always get competing offers.** Best negotiation leverage.
- **Never name a number first.** "What is the budgeted range for this role?"
- **Counter offer is expected.** They lowball you by 15-20%.
- **Total comp > base.** Stocks, signing bonus, learning budget.
- **Be ready to walk away.** The best negotiation stance is being okay with no.

**Realistic 20 LPA offer at 3 YOE in AI:**
- Base: 15-18 LPA
- Stocks: 2-4 LPA/year
- Bonus: 1-2 LPA
- **Total: 20-24 LPA**

## ⚠️ Common job hunt mistakes

1. ❌ **Mass applying on Naukri.** 5% response rate. Waste of time.
2. ❌ **No portfolio.** "I took Andrew Ng's course" is not a portfolio. Ship projects.
3. ❌ **Not networking.** 50%+ of jobs at 20+ LPA are filled via referral.
4. ❌ **Quitting too early.** Job hunt takes 3-6 months. Plan for it.
5. ❌ **Negotiating with one offer.** Always have 2+ in pipeline.
6. ❌ **Undervaluing yourself.** If you can do all of the above, you're worth 20+ LPA.

## 🎯 The 90-day job hunt sprint (if you want to compress)

If you want to compress to 6-9 months total:
- Cut Phase 5-6 content to 50% (just learn the must-knows, skip nice-to-haves)
- Use existing projects from your current job if they relate (with company permission)
- Start interviewing at month 5-6 even if you feel unready
- Apply to 10+ companies/week
- Use referrals aggressively

**Risk:** May land at 15-18 LPA instead of 20+. But you can renegotiate or jump again in 12-18 months.
