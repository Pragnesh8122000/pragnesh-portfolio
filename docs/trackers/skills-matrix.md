## How to use

Rate yourself 1-5 in each skill:
- **1** = never heard of it
- **2** = heard of it, can't use it
- **3** = can use it with help / docs
- **4** = comfortable, can ship with it
- **5** = could teach others, can debug deep issues

Target = what you need to land a 20+ LPA AI Engineer role.

## 💡 Self-assessment tips

1. **Be honest.** Inflating scores kills your progress. "I'm a 2 in PyTorch" is a great starting point.
2. **Rate based on shipping, not knowing.** Can you BUILD a thing, or just talk about it?
3. **Re-rate quarterly.** What was a 3 last month should be a 4 if you've used it a lot.
4. **3 = "I can ship a small project with it."** That's the bar.
5. **5 = "I could write a book chapter on this."** Rare. Don't aim for 5 in everything.
