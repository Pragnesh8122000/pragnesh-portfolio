## ✅ The 5 rules of this roadmap

1. **Build in public.** Every project goes on GitHub with a README. Post about it on LinkedIn.
2. **2 hours/day. No zero days.** Even 30 minutes counts. Consistency beats intensity.
3. **Ship, don't study.** 70% building, 30% learning. Don't get stuck in tutorial hell.
4. **Network > Skills.** 50% of jobs at 20 LPA come through referrals. Start DMing people now.
5. **Track everything.** If you don't write it down, you didn't do it.
